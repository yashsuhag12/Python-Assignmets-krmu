Daily Calorie Tracker CLI

This project was created as a mini-assignment for the session:
Programming for Problem Solving using Python.

Files included:
1. tracker.py  — main Python script.
2. calorie_log_exactly_met.txt  — run where calories matched limit.
3. calorie_log_within_limit.txt — run where calories were below limit.
4. calorie_log_exceeded.txt     — run where calories exceeded limit.

Each log file contains timestamp, meal list, total calories, average, and result.
